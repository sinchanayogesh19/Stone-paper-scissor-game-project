# Stone Paper Scissors Game

## Overview

The Stone Paper Scissors Game is a simple interactive game developed using programming concepts where a player competes against the computer. The computer randomly selects Stone, Paper, or Scissors, and the winner is determined based on the standard game rules.

## Features

* User-friendly interface
* Random computer choice generation
* Real-time result display
* Score tracking for player and computer
* Multiple rounds gameplay
* Simple and lightweight application

## Technologies Used

* Python
* Tkinter (for GUI) / HTML, CSS, JavaScript (for Web Version)
* Random Module

## Game Rules

1. Stone beats Scissors.
2. Scissors beats Paper.
3. Paper beats Stone.
4. If both player and computer choose the same option, the game ends in a draw.

## How to Play

1. Run the application.
2. Select either Stone, Paper, or Scissors.
3. The computer will automatically generate its choice.
4. The result (Win, Lose, or Draw) will be displayed.
5. Continue playing multiple rounds and track your score.
